# DAG Visualizer

React-приложение для визуализации направленных графов (DAG)

## Функционал
- Визуальное построение графа
- Экспорт в PNG
- Сохранение/загрузка графов

## Запуск
```bash
# Установка
npm install

# Запуск сервера (отдельный терминал)
node server/server.js

# Запуск фронтенда
npm start
```
